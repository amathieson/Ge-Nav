• Why does the stop count change?
    The stop count displayed on the Stops page is different on different days because TPG occasionally close some of their stops. It can also change during the day if they send the wrong data to our servers.
  • Search functions
    In the search box you can search by stop code by entering "stopCode:" and then the stop code in the search box. You can also search by line by entering "line:" and them the line code into the search box.
